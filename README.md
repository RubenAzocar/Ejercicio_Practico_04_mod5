# Ejercicio Práctico 04 - Módulo 5: Consultas SQL sobre Series de Netflix

Este ejercicio te permite practicar la creación y manipulación de una tabla de series de Netflix utilizando PostgreSQL y consultas SQL.

## Contenido

- **Creación de la tabla**: Estructura para almacenar información de series.
- **Inserción de datos**: Agrega varias series con diferentes géneros, temporadas y años de estreno.
- **Consultas de selección**: Filtra y ordena series según distintos criterios.
- **Consultas de agregación**: Calcula promedios, máximos y mínimos.
- **Eliminación y reinserción**: Borra y recupera datos según condiciones específicas.

## Instrucciones

1. Abre el archivo `Series_Netflix.sql` en tu gestor de base de datos PostgreSQL.
2. Ejecuta cada bloque de código SQL para crear la tabla, insertar datos y realizar las consultas.
3. Lee los comentarios en el archivo para entender el propósito de cada consulta.
4. Puedes modificar o agregar nuevas series para experimentar con los resultados.

## Consultas incluidas

- Listar series según cantidad de temporadas.
- Encontrar el año de la serie más antigua y más nueva.
- Calcular promedios de año de estreno y temporadas.
- Filtrar series por nombre o rango de años.
- Eliminar y reinsertar registros según condiciones.
